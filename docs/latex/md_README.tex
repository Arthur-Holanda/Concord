O código em C++ apresenta uma implementação básica de um sistema de chat em um servidor. Ele é dividido em várias classes, cada uma responsável por um aspecto específico do sistema.\hypertarget{md_README_autotoc_md2}{}\doxysubsection{Compilando}\label{md_README_autotoc_md2}
Este projeto pode ser compilado com o CMake.

Para compilar este projeto com o CMake, siga estas etapas (a partir da pasta raiz)\+:


\begin{DoxyEnumerate}
\item {\ttfamily cmake -\/S source -\/B build}\+: solicita ao CMake que crie o diretório de compilação e gere o Makefile Unix com base no script encontrado em source/\+CMake\+Lists.\+txt, no nível atual.
\item {\ttfamily cmake -\/-\/build build}\+: inicia o processo de compilação que cria os dois alvos (executáveis) dentro de build.
\end{DoxyEnumerate}

O executável é criado dentro do diretório {\ttfamily build} que caso não tenha sido criada antes, é criada no momento do comando {\ttfamily cmake -\/S source -\/B build}.

Para executar os testes, você tem duas opções\+:


\begin{DoxyEnumerate}
\item Acesse o diretório {\ttfamily build} com o comando cd build e execute os testes com {\ttfamily ./\+Concord}.
\item Execute os testes diretamente com o comando {\ttfamily ./build/\+Concord} a partir da pasta raiz do projeto.
\end{DoxyEnumerate}

O diretorio contem um script de teste, caso queira o testar, use {\ttfamily ./build/\+Concord \texorpdfstring{$<$}{<} script.\+txt} na pasta raiz do projeto. \hypertarget{md_README_autotoc_md3}{}\doxysubsection{Comandos}\label{md_README_autotoc_md3}
